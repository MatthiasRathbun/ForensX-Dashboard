How to use IPGeo

Before running on your local computer, make sure you have python 3.6+ with the latest version of pandas installed and Hash.key in the same folder as the python file.
To Install Pandas, run pip install pandas.
Before running for the first time, make a folder named "results". This will have all the .json and .csv files stored in it after running.
To query IP, run python request.py in your command prompt environment with the pandas installed.
You will be prompted to enter how many IP searches you want to make.
Your prompt will look like this How many IP searches would you like to make?
You will be prompted to enter the IP you want to search. As an example use 8.8.8.8 which is Google's IP.
You will be given a prompt like this Please Enter IPv4 request in x.x.x.x format. 8.8.8.8
After entering the IP, a file called result.csv and result.json are saved which have the relevant information regarding your IP Search.
